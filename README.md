# Verilog-Digital-Design

A structured collection of fundamental digital design implementations using Verilog HDL.  
This repository focuses on building a strong foundation in combinational and sequential circuit design through synthesizable RTL modules and simulation-based verification.

## Repository Structure

| Folder | Description |
|---|---|
| `01_Combinational` | Logic gates, multiplexers, decoders, encoders, adders, subtractors, comparators, and arithmetic circuits |
| `02_Sequential` | Flip-flops, latches, counters, shift registers, sequence detectors, and finite state machines (FSMs) |
| `03_Verification` | Verilog testbenches for functional verification and simulation |
| `04_RTL_Designs` | Full RTL Module of the design |
| `05_Waveforms` | Output Waveform of the design |

## Tools & Workflow

- Verilog HDL
- Xilinx Vivado Design Suite
- XSim Simulator
- RTL Design & Simulation
- Synthesis and Timing Analysis
- Waveform Verification
- FPGA Development Flow
